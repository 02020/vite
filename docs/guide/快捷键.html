vscode

  1. Ctrl + Shift + O 查找 → 可以用关键字过滤符号
  2. Ctrl + P 文件查找。快速打开文件列表
  3. 折叠代码： Ctrl + Shift + [/]
  4. Shift + Alt + F 格式化
  5. Shift + Alt + Up/Down 复制行
  6. 逐个选择文本 Ctrl + Shift + Right
  7. Alt + Left/Right 前进或者后退到光标所在位置
  8. Expand Bracket Selection 自动选择整个块，从开始的大括号到结束。
  9. F12 跳转到定义
  10. Alt + F12 以预览方式在当前页面显示定义
  11. Shift + F12 查看光标所在函数或变量的引用

光标操作

  1. Ctrl + Alt + Up/Down
  2. Ctrl + D 选中相同的
  3. Shift + Alt + 鼠标
